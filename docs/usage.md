# Usage
> This documentation applies to brutecraber v0.6.0

> This project is under active development

## Running the tool

After installation, you can run the tool using the binary

```bash
./brutecraber

brutecraber

./target/release/brutecraber
```

By default, the tool will display some help for the user using the next command

```bash
./brutecraber --help
```

### Command line options

-f -> Path to the file that have the hashes that you want to crack

-w -> Wordlist to do the bruteforce to your hashes

-t -> Put the specific hash to crack (the aplication have a automatic detection system but can fail)

-V -> See the current version
